SHARON

Spinners & page loaders Collection






Install

Copy <script>window.onload = function(){ document.querySelector(".preloader").style.display = "none"; }</script> or

Copy Javascript code from https://xbrpw.github.io/sharon/preloader.js and paste into text editor and save sharon.js

Copy CSS code from https://xbrpw.github.io/sharon/preloader.css and paste into text editor and save preloader.css




How to do

Add css file link into your html file:

<link rel="stylesheet" href="https://xbrpw.github.io/sharon/preloader.css"/>

Add js

<script>window.onload = function(){ document.querySelector(".preloader").style.display = "none"; }</script>
<script src="https://xbrpw.github.io/sharon/preloader.js"></script>

Add responsible meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
